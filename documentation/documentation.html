Files

You can see here the files and libraries that used in Haroshe.

    CSS Files

  • Animate.css for using scroll to reveal effects.
  • bootstrap.css for bootstrap styles.
  • lightbox.css for lightbox styles in Gallery section.
  • responsive.css for responsive fixes.
  • style.css the main style of Haroshe.

    • JS Files

  • boostrap.jsfor bootstrap functions.
  • jquery.jsfor jquery functions.
  • lightbox.jsfor lightbox functions to animate images in gallery section.
  • wow.jsthe jquery library for scroll to reveal divisions.
